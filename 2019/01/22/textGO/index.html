textGO 可以帮您提取图片中文字的app

textGO 是 5km(十里) 开发的一款 ocr 工具，目前只开发了 macOS 版，它可以帮您快速提取图片中的文字，方便您的工作。

下载

点击下面图标下载最新版：

特性

  • 依赖大公司的 OCR 引擎，识别准确，配置自己的 api_key 和 secret_key 获得足够的免费使用次数(50000次/日)
  • 轻量，占用系统资源少
  • 一款状态栏小工具，操作便捷；自然的操作逻辑，顺畅识别
    • 截图 OCR
    • 拖动图片到状态栏图标进行OCR
    • 识别结果推到系统剪切板
  • 界面元素跟随系统，完美适应 macOS 下的 Dark 和 light 模式

使用教程

相信您已经下载了最新的 textGO，下面简单说一下如何使用。

获取百度 AI 文字识别应用的密钥

因为 app 设计的初衷就是，让使用者获得准确的 OCR 结果的同时，能够有尽量多的使用次数，所以作者首先选择了使用 百度 AI 提供的 OCR 服务，所以在使用 textGO 之前需要获取一下属于您自己的key，这样您就可以每天独用 50000次了。

百度账号

首先您得有百度账号，相信大部分人都有了自己的百度账号，没有的话可以访问百度注册。

百度 AI 控制台

打开百度 AI 首页，点击右上角的 [控制台] 按钮，使用您的百度账号登录，成功后就进入了控制台。

创建文字识别应用

  1. 点击控制台左侧的 文字识别 就会进入文字识别的概览页面

  2. 可以看到 应用 区，点击 创建应用

  3. 填写应用名称和描述即可，其它默认：

  4. 点击立即创建，创建成功后如下：

  5. 点击返回应用列表 ，就能看到自己刚创建的应用的详细信息，其中主要关注的是 API key 和 Secret Key ，在 textGO 中会用到。

配置 textGO

  1. 运行最新的 textGO，会在状态栏看到一个小logo：

  2. 点击它就会看到菜单项：

  3. 点击 偏好设置，就会打开设置窗口，在窗口中输入上小节得到的 API key 和 Secret Key 后点击保存，此时app就可以正常使用了。

使用

目前有两种使用方式，最后都是将结果设置到剪切板上，要使用结果，直接在您的文档中粘贴就行。
